# CameraScanner SDK — Android Devlog

---

## Origin

Windows Python SDK ported to Android. Architecture and detection philosophy carried over; code rewritten from scratch in Kotlin. Target segments (Web3/airdrop, dating, PvP) are mobile-first.

---

## Project Setup

Android Studio Panda 4 | Min SDK: API 26 | Kotlin DSL

---

## v1.0.0 — Initial Port

4 detection methods and a single public entry point established.

- METHOD 1: Checks whether known virtual camera packages are installed.
- METHOD 2: Analyzes camera hardware characteristics.
- METHOD 3: Scans the kernel filesystem for virtual camera drivers.
- METHOD 4: Checks system alert window permission and active processes.
- Entry point: `isFakeCamera()`. Returns: `isFake`, `confidence` (HIGH / MEDIUM / CLEAN), `detected`, `errors`.
